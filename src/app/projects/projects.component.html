Pre-built API CI/CD pipeline setup with GitHub Actions for WSO2 API Cloud Users
Technologies:

Shell Script / Docker/ GitHub Actions / Git / Linux tools / APIMCLI

Source Code

This project introduces a ready-to-use workflow setup that enables client managed CI/CD for WSO2 API Cloud or any distributed deployment of WSO2 API Manager hosted publicly. This approach eliminates the CI/CD burden on WSO2 API Cloud or deployment administrators and also allows clients to have full control of CI/CD with the use of workflows as required.

The speciality of this project is that clients will be able to manage a CI/CD pipeline for their API projects without having to configure any third party software, libraries, packages (Java/npm/newman/curl/...etc), and would not even require WSO2 APIM tools such as APIMCLI or APIMCTL. Marketplace
